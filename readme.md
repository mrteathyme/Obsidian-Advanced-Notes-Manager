# Obsidian Advanced Notes Manager

Automatically create, rename, move and delete notes based on configurable rules.

## Highlighted Features
- Automatically move a file to a folder based on a ruleset (Currently only moves new files to a folder of Current Date and an Attachments sub-folder in that folder for non markdown files)
- You tell me? Im expanding this with features as I need them but im always open to requests.
